# Sandesh Gorde — Portfolio

Personal portfolio site, hand-built from scratch to learn frontend for fun ! 

## Stack
- [Astro](https://astro.build) — site framework
- [Vue](https://vuejs.org) — interactive components
- [UnoCSS](https://unocss.dev) — atomic CSS
- TypeScript

## Development

```sh
npm install
npm run dev
```
